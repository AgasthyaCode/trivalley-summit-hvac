# Summit Air & Heat Website

Static HVAC company website with AI-generated placeholder photos.

## Files

- `index.html` - page content and sections
- `styles.css` - responsive design
- `script.js` - mobile navigation and demo form behavior
- `assets/images/` - replaceable website photos. The `.jpg` files are used by the site; the `.png` files are the generated originals.
- `DEPLOYMENT.md` - live deployment options

## Replace These Placeholders

- Company name: `Summit Air & Heat`
- Phone number: `(555) 014-3728`
- Email: `service@summitairheat.example`
- License number: `HVAC-000000`
- Service area city names
- Maintenance plan prices
- Reviews and customer names
- AI-generated photos in `assets/images/`

Open `index.html` in a browser to preview locally.
